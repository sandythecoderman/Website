## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/sandythecoderman/Website.git
   cd Website
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run locally**
   ```bash
   npm run dev
   ```
   Or use Python:
   ```bash
   python3 -m http.server 8080
   ```

4. **Run tests**
   ```bash
   npm test
   ```
